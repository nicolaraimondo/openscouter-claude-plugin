# OpenScouter Claude Code marketplace

A one-plugin Claude Code marketplace for [OpenScouter](https://openscouter.com).

```
/plugin marketplace add nicolaraimondo/openscouter-claude-plugin
/plugin install openscouter@openscouter
```

This installs the **openscouter** plugin: the OpenScouter MCP connection, the
`/openscouter` command, and a skill that guides Claude Code through your
usability studies. See [`plugin/README.md`](plugin/README.md).

> This directory is the source of truth for the plugin. It is published to the
> public `nicolaraimondo/openscouter-claude-plugin` repo for client installation. The
> canonical skill and command also live at the repo root (`skills/openscouter`,
> `commands/openscouter.md`); keep them in sync.
